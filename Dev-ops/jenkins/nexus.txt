script for Nexus:
==========================

touch script.sh
chmod u+x script.sh

vi script.sh

echo "installation starts ..."
sleep 2
sudo apt update && sudo apt upgrade -y
sleep 2
cd /opt
sleep 2
sudo wget https://download.sonatype.com/nexus/3/nexus-3.90.2-06-linux-x86_64.tar.gz
sleep 2
sudo tar -xzvf nexus-3.90.2-06-linux-x86_64.tar.gz
sleep 2
sudo rm -rf nexus-3.90.2-06-linux-x86_64.tar.gz
sleep 2
sudo mv nexus-3.90.2-06/ nexus
sleep 2
sudo useradd nexus
sleep 2
cd nexus/bin/
sudo touch nexus.rc
sleep 2
echo 'run_as_user="nexus"' | sudo tee /opt/nexus/bin/nexus.rc
sleep 2
sudo chown -R nexus:nexus /opt/nexus
sleep 2
sudo chown -R nexus:nexus /opt/sonatype-work/
echo "installation success"

========================================


go to: /etc/systemd/system
	cd /etc/systemd/system
Create a nexus system unit file to auto start nexus server on startup 
	reference -> https://help.sonatype.com/en/run-as-a-service.html
	sudo touch nexus.service
	sudo vi nexus.service

Add following contents to nexus.service
========================================

[Unit]
Description=nexus service
After=network.target

[Service]
Type=forking
LimitNPROC=65536
LimitNOFILE=65536
ExecStart=/opt/nexus/bin/nexus start
ExecStop=/opt/nexus/bin/nexus stop

User=nexus
Restart=on-abort
TimeoutSec=600

[Install]
WantedBy=multi-user.target

========================================

Add nexus service to boot and start nexus:

	sudo systemctl daemon-reload
	sudo systemctl enable nexus.service
	sudo systemctl start nexus.service
